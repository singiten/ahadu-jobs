# አሐዱ ሥራ - Ahadu Jobs

![Ahadu Jobs Banner](https://via.placeholder.com/1200x300/1E3A8A/FFFFFF?text=Ahadu+Jobs+-+Your+One+Opportunity)

Ahadu Jobs is a full-stack job portal platform connecting Ethiopian job seekers with employers. The name "Ahadu" (አሐዱ) means "The One" or "Unique" in Amharic, representing the perfect match between talent and opportunity.

## ✨ Features

### 👤 For Job Seekers
- ✅ User registration and login with JWT authentication
- ✅ Browse jobs with advanced filters (category, location, salary, job type)
- ✅ Search by keywords, company, or location
- ✅ Save jobs to favorites
- ✅ Apply to jobs with resume upload (PDF/DOC)
- ✅ Track application status in real-time
- ✅ Personal dashboard with application statistics
- ✅ Email notifications for application updates

### 🏢 For Recruiters
- ✅ Company registration and profile management
- ✅ Post and manage job listings
- ✅ View applicants for each job
- ✅ Update application status (pending, reviewed, shortlisted, hired, rejected)
- ✅ View detailed applicant information
- ✅ Dashboard with job performance metrics
- ✅ Email notifications for new applications

### 🛡️ For Everyone
- ✅ Role-based access control (seeker, recruiter, admin)
- ✅ Secure password hashing with bcrypt
- ✅ JWT token authentication
- ✅ Responsive design for mobile and desktop
- ✅ Modern professional UI with animations
- ✅ Toast notifications for user feedback

## 🛠️ Tech Stack

### **Frontend**
- React 18 with Hooks
- React Router 6 for navigation
- Context API for state management
- Axios for API calls
- React Toastify for notifications
- CSS3 with custom animations

### **Backend**
- Node.js with Express
- MongoDB with Mongoose ODM
- JWT for authentication
- bcrypt for password hashing
- Nodemailer for email notifications
- Multer + Cloudinary for file uploads
- Node-cron for scheduled jobs

### **DevOps & Tools**
- Git for version control
- npm for package management
- nodemon for development
- Ethereal for email testing
- Thunder Client/Postman for API testing
